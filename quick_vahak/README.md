# 🚚 Quick Vahak - Full Stack Logistics Solution
A high-performance transport management application built with *Next.js*.

## 🚀 Key Features
* *Full-Stack Logic:* Integrated backend logic with Next.js API routes for a unified developer experience.
* *Server-Side Power:* Optimized performance using Server-Side Rendering (SSR).
* *Modern Styling:* Built with *Tailwind CSS* for a responsive, mobile-first design.
* *Deployment:* Successfully deployed and live on Vercel.
