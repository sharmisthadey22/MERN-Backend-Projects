# Blog Management System

A full-stack blog application that allows users to create, read, and manage blog posts. This project focuses on structured data handling and backend logic.

## 🚀 Key Features
* *CRUD Functionality:* Users can Create, Read, Update, and Delete blogs.
* *Server-side Logic:* Built with Express.js to handle complex routing.
* *EJS/Views:* Managed front-end rendering using views folder.
* *Data Persistence:* Integrated with MongoDB for reliable storage.

## 🛠️ Tech Stack
* *Language:* JavaScript
* *Backend:* Node.js, Express.js
* *Database:* MongoDB
* *View Engine:* EJS / HTML

## ⚙️ Setup
1. Clone the repo.
2. Run npm install.
3. Start using node index.js.
